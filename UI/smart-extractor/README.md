# Smart Extractor

Base limpia de Nuxt 4 con Nuxt UI, TypeScript y ESLint.

## Scripts

```bash
pnpm dev
pnpm build
pnpm preview
pnpm lint
pnpm typecheck
```

## Estructura base

- `app/app.vue`: shell principal de la aplicación.
- `app/pages/index.vue`: página inicial mínima.
- `app/assets/css/main.css`: entrada global de estilos.

## Inicio

```bash
pnpm install
pnpm dev
```

La app quedará disponible en `http://localhost:3000`.
